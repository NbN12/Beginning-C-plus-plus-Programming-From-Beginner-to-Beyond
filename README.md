## Prerequistes
* g++ (mingw for windows)
## Setup
1. Install g++ (mingw)
2. Add mingw(g++)-path\bin to env (not in env.Path) which name it MINGW
3. Run using **Run build Task** in vscode or (Ctrl+Shift+B) and choose "build and run"
